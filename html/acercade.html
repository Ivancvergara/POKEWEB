Objetivo de la Página Web

El proyecto es requerimiento indispensable para culminar el Bootcamp de Frontend por Código Facilito y recibir un certificado digital por haber concluido satisfactoriamente el programa, sin embargo, el beneficio más grande es el que traerá para la carrera profesional.

El proyecto busca que se comience a construir un portafolio para que se pueda presentar al aplicar para empresas que busquen Frontend Developersk
